Query Patterns

Các truy vấn chính được tối ưu hóa theo data model design

Trip History Playback

Phát lại lịch sử hành trình

SELECT * FROM trip_locations
WHERE trip_id = ?
ORDER BY timestamp ASC;

Use case: Phát lại hành trình trên bản đồ

Performance: Clustering key timestamp DESC giúp query nhanh

Compliance: Điều tra tranh chấp, tuân thủ pháp luật

Get Latest Location

Vị trí mới nhất của tài xế

SELECT * FROM trip_locations
WHERE trip_id = ?
LIMIT 1;

Use case: Hiển thị vị trí hiện tại trên UI

Performance: O(1) operation, không scan toàn bộ data

Optimization: Clustering key DESC ensures điểm mới nhất ở đầu

Driver Performance Analytics

Phân tích hiệu suất tài xế

SELECT * FROM driver_analytics
WHERE driver_id = ?
AND week_start_date >= ?;

Use case: Dashboard hiệu suất theo tuần

Analytics: Tổng khoảng cách, số chuyến, vi phạm tốc độ

Pattern: Clustering key week_start_date DESC supports "xem tuần gần đây"

Get Alerts

Lịch sử cảnh báo

SELECT * FROM alerts
WHERE driver_id = ?
AND timestamp >= ?;

Use case: Xem cảnh báo speeding & proximity

Monitoring: Hỗ trợ điều tra vi phạm

Pattern: Alert mới nhất luôn ở đầu (timestamp DESC)

Apache Cassandra - NoSQL Database

Distributed database optimized for high write throughput and time-series data

Khái niệm

NoSQL Database

Đặc điểm chính

Distributed - Phân tán dữ liệu

Masterless - Không SPOF

Time-series optimized

High availability

Kiến trúc Peer-to-Peer

Mọi node đều equal - Data partitioned & replicated

Tại sao chọn?

4 lý do chính

Write Throughput

1M+/s

Masterless architecture, phù hợp 17 GB GPS data/ngày

Horizontal Scaling

Thêm node = tăng capacity tuyến tính, tự động rebalancing

Time-Series Optimized

Clustering key by timestamp, tối ưu range scan theo thời gian

No SPOF

Tự động failover, data replicated trên nhiều nodes

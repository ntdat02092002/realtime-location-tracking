Đánh giá Cassandra vs PostgreSQL cho bài toán GPS Tracking với write-heavy workload

Apache Cassandra

NoSQL Database (Đã chọn)

TỐI ƯU

Ưu điểm chính

  • 1M+ writes/sec - Masterless architecture
  • Time-series optimized - Clustering key by timestamp
  • Horizontal scaling - Linear scalability
  • No SPOF - Auto failover
  • Schema flexible - Add columns without downtime

Phù hợp cho Big Data

• 86.4 triệu GPS points/ngày (17 GB)

• Time-series queries cần tốc độ cao

• Horizontal scaling dễ dàng

PostgreSQL

Relational Database

Hạn chế chính

  • 10k-50k writes/sec - Single primary bottleneck
  • Vertical scaling - Cần máy chủ lớn, tốn kém
  • Manual sharding - Phức tạp, dễ lỗi
  • ALTER TABLE locks - Block writes trên large tables

Phù hợp cho OLTP

• ACID transactions đảm bảo tính toàn vẹn

• Complex joins và relational queries

• Ứng dụng cần tính nhất quán cao

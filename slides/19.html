DSL vs Processor API

DSL (Domain-Specific Language)

API khai báo cấp cao, dễ sử dụng

filter map join aggregate windowedBy

Processor API

API cấp thấp, control chi tiết

• Direct access State Store

• Custom logic per record

• stateStore.get(key)

• stateStore.put(key, value)
